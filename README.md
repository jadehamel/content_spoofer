# content_spoofer
A javascript library to extract content from other websites. Link to full size pictures, title, descriptions.

Use `spoof` Object to get all the classes

- Amazon Canada

路 Get fullsize from thumbs alternatives images on Amazon Canada when on a product page

`Amazonca.getthumbs`

路 Get product Amazon Canada Id

`Amazonca.product_id`    

路 Get Amazon Canada product title (Without Amazon mention)

`Amazonca.title`

路 Get Amazon Canada product description (Without Amazon mention)

`Amazonca.description`
